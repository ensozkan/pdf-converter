# PDF Dönüştürücü

Dosyaları sürükle-bırak ile PDF'e çeviren masaüstü uygulaması.

## Desteklenen Formatlar

| Format | Uzantılar |
|--------|-----------|
| Resimler | JPG, PNG, WEBP, GIF, BMP, TIFF |
| Metin | TXT, MD |
| Word | DOCX |
| Sunum | PPTX, PPT (LibreOffice gerekli) |
| Web | HTML, HTM |
| Veri | CSV, XLSX, XLS |

## .exe Nasıl İndirilir?

1. Bu sayfada sağ üstte **Actions** sekmesine tıkla
2. Soldaki listeden **Windows EXE Derle** iş akışını seç
3. En üstteki (en son) çalıştırmaya tıkla
4. Sayfanın altında **Artifacts** bölümünde **PDF-Donusturucu-Windows** yazar
5. Tıkla, ZIP indir, içinden `PDF Donusturucu.exe` çıkar, çalıştır

> ⚠️ PPTX dönüşümü için LibreOffice kurulu olmalı: https://www.libreoffice.org
